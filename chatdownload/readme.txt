To run this chat application .
(1).first double click the "runserver" batch file.
this will start the server program.
(2).Then you double-click the "runclient" batch file.
this will start the client program.
you can run as many as client programs.
to run all these programs on different machines,you need to do slight change in the source file.just change the IP address according to where you are going to put your server program.
